# 📱 App Insights & User Analytics Dashboard

## 📌 Overview

This project presents an interactive **Power BI dashboard** to analyze Google Play Store app data. It provides insights into app installs, reviews, content ratings, categories, and user preferences to help understand market trends and app performance.

## 🎯 Objectives

* Analyze app performance across categories.
* Compare free vs paid applications.
* Identify top-performing apps based on installs and reviews.
* Explore content ratings and genre distribution.
* Build an interactive dashboard for business insights.

## 📊 Dashboard Features

* KPI Cards (Total Apps, Total Installs, Total Reviews, Average Rating, Total Categories)
* Top Categories by Installs
* Top Reviewed Apps
* Free vs Paid Apps Distribution
* Content Rating Distribution
* Installs by Category (Treemap)
* Interactive Slicers (Category, Type, Content Rating, Genres)

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX
* Microsoft Excel

## 📈 Key Insights

* Game and Communication categories have the highest installs.
* Free apps dominate the Play Store.
* Most apps are rated **Everyone**.
* Instagram is among the most reviewed applications.
* A few categories account for the majority of total installs.

## 📂 Dataset

* Google Play Store Apps Dataset
* Google Play Store User Reviews Dataset
